Home Page

Overview of all the tools available for monitoring CTHD (Current Total Harmonic Distortion) within the GUI (Graphics User Interface).

Server Map Page

Information on all existing Data Center servers. Including a map that shows the entire Data Center server floor layout and number of server rows. From this page the user can select a specific server row, which will allow the user to see the CTHD Data for each server rack and Graph showcasing the history of the Data Center server rack CTHD.
